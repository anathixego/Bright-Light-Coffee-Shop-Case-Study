Bright Light Coffee Shop Sales Analysis

Project Overview

This project explores sales data for Bright Light Coffee Shop to uncover key business insights and support decision-making.
It focuses on understanding sales performance, top products, and peak operating hours using SQL, Canva and Excel.


Objectives

Identify top 5 selling products
Analyse revenue by store location and product
Track peak sales times to plan staffing and operations
Create interactive dashboards for management decisions

Dataset Information

Dataset: shoping_trends.csv

Main Columns:

Product — item sold
Quantity — number of items sold per transaction
Store Location — branch of the coffee shop
Revenue — total sale amount
Transaction Time — time of purchase


Tools & Technologies

Canva – for visualization and dashboards

SQL (Snowflake) – for data analysis queries

Excel – for data cleaning and quick calculations

Key Insights

Top-selling products drive the majority of total revenue
Sales peak during mid-morning and evening in one store
Certain store locations consistently outperform others

Skills Demonstrated

Data Cleaning & Transformation (Microsoft Excel)
SQL Query Writing (Snowflake)
Data Visualization and Presentation (Canva)
